# Verificador Título Eleitor 🗳️

Um verificador de título de eleitor desenvolvido em **C#** que valida e processa dados de títulos eleitorais com base em regras específicas.

## 📋 Sobre o Projeto

Este projeto foi criado como atividade educacional com o objetivo de implementar uma classe verificadora que valide títulos de eleitor de acordo com regras pré-estabelecidas, aplicando conceitos de programação orientada a objetos.

## 🎯 Objetivo

Fornecer uma solução que:
- Valide a estrutura de títulos eleitorais
- Aplique regras de verificação específicas
- Demonstre boas práticas de programação em C#
- Serve como ferramenta educacional

## 🛠️ Tecnologias

- **C#** - Linguagem de programação principal
